Leaf inclination distribution function.

Expression
  • Planophile:

    \[ F(\theta) = \frac{2}{\pi}\left[\theta + \frac{1}{2}\sin{2\theta}\right] \]

  • Erectophile:

    \[ F(\theta) = \frac{2}{\pi}\left[\theta - \frac{1}{2}\sin{2\theta}\right] \]

  • Plagiophile:

    \[ F(\theta) = \frac{2}{\pi}\left[\theta - \frac{1}{4}\sin{4\theta}\right] \]

  • Extremophile:

    \[ F(\theta) = \frac{2}{\pi}\left[\theta + \frac{1}{4}\sin{4\theta}\right] \]

  • Spherical:

    \[ F(\theta) = 1 - \cos{\theta} \]
